Teknologi & Sistem Pengembangan

Bagian ini menjelaskan sistem yang digunakan untuk memastikan pengembangan terukur dan mudah dikembangkan ke fase berikutnya.

Teknologi Utama
  • Framework: Yii2 (PHP)
  • Database: MySQL
  • UI: AdminLTE (Enterprise dashboard style)
  • Deployment: Server/VPS atau on-premise (sesuai kebijakan klien)
Prinsip Keamanan & Kontrol
  • Login & hak akses berbasis role (Admin/HC/Manajemen)
  • Audit trail aktivitas (timeline/catatan aktivitas)
  • Validasi input data & upload dokumen
  • Backup database & versioning source code
Kebutuhan Infrastruktur (Rekomendasi Minimum)
  • CPU 2 vCPU
  • RAM 4 GB
  • Storage 40 GB SSD
  • Linux Server (recommended)
  • Web server: Nginx/Apache
  • SSL untuk akses aman

Spesifikasi dapat ditingkatkan sesuai jumlah user dan volume dokumen.
